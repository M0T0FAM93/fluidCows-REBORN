# Contributing to Fluid Cows Reborn

Thank you for your interest in contributing to Fluid Cows Reborn.

Contributions are welcome and appreciated.

---

## Before You Start

• Make sure you are using the supported Minecraft and NeoForge version  
• Test your changes in a local development environment  
• Keep changes focused and well scoped  

For large changes or new features please open an issue first to discuss the idea.

---

## How to Contribute

1 Fork the repository  
2 Create a new branch for your change  
3 Make your changes and test them  
4 Commit with a clear message  
5 Open a pull request  

---

## Code Guidelines

• Follow the existing code style and formatting  
• Keep code readable and documented when needed  
• Avoid breaking compatibility unless discussed first  
• Do not include generated or compiled files  

---

## Bug Reports and Feature Requests

• Use the issue tracker for bugs and feature ideas  
• Include logs and reproduction steps when reporting bugs  
• Clearly describe the intended behavior for new features  

---
